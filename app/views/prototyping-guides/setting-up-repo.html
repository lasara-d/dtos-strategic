Setting up a new repo with the NHS prototype kit code

1. Create a new repository on the GitHub NHS.UK space

Go to the NHS.UK GitHub space and create a new public repository.

2. Navigate to the folder where you keep your prototypes

Use the terminal to navigate to the folder where you would save your prototypes, for example, your home folder. To do this, run:

cd name of folder where you keep your prototypes

3. Clone your new repository to your local machine

Visit your repository on GitHub and copy the URL it gives you. Then run this command:

git clone url of the repo you created

4. Download and extract the prototype kit files

Navigate into the folder of the repository you just cloned:

cd folder name

Download and extract the prototype kit files.

Run the following commands or download a zip of the prototype kit files through your web browser / github.com and extract the files into place.

{{ warningCallout({ "heading": "Important", "HTML": "

If copying files over manually from the prototype kit folder, be careful to retain the hidden 'dotfiles' such as .gitignore. Press 'Command Shift .' on a Mac to see these and make sure you copy them over too.

curl -LO https://github.com/nhsuk/nhsuk-prototype-kit/archive/refs/heads/main.zip 
unzip main.zip
mv nhsuk-prototype-kit-main/* .
rm -rf nhsuk-prototype-kit-main
rm main.zip

5. Use the terminal to add the changes

Run this command:

git add -A

6. Use the terminal to commit the changes

Run this command:

git commit -m "Initial commit of prototype files"

7. Push the changes

Run this command:

git push
